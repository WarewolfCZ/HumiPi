# HumiPi
Home humidity and temperature sensor with eInk display

This project is using Adafruit DHT library (https://github.com/adafruit/Adafruit_Python_DHT) and e-Paper library (https://github.com/waveshare/e-Paper)

You need to install Adafruit_DHT for HumiPi to work:
> sudo pip3 install Adafruit_DHT
